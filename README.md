# Virtual UI 🚀

Virtual UI is an AI-powered React UI Library and SaaS platform built using the MERN stack. Developers can install prebuilt UI components through NPM or generate custom React components dynamically using AI prompts.

🌐 Live Demo: https://virtual-ui-1-g2i2.onrender.com/

🔗 Repository: https://github.com/Nitesh-kumar-jha-2002/Virtual_UI.git

---

# 🌟 Features

- 🤖 AI-Based UI Component Generation
- 📦 Publish & Use NPM React Library
- 💳 Razorpay SaaS Payment Integration
- 🔐 Authentication & User Management
- ⚡ Dynamic Prompt-Based UI Generation
- 🎨 Modern Responsive UI
- ☁️ Full MERN Stack Architecture
- 🔥 Real-World SaaS Project Structure

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Nitesh-kumar-jha-2002/Virtual_UI.git
cd Virtual_UI
```

---

## 2️⃣ Install Dependencies

### Frontend

```bash
cd virtual-ui-client
npm install
```

### Backend

```bash
cd virtual-ui-server
npm install
```

### UI Library

```bash
cd virtual-ui-lib
npm install
```

---

# 🔑 Environment Variables

## Server `.env`

```env
PORT=8000

MONGODB_URL=your_mongodb_connection

JWT_SECRET=your_secret_key

OPENROUTER_API_KEY=your_openrouter_api_key

RAZORPAY_KEY_ID=your_razorpay_key
RAZORPAY_KEY_SECRET=your_razorpay_secret
```

---

## Client `.env`

```env
VITE_FIREBASE_APIKEY=your_firebase_api_key

VITE_RAZORPAY_KEY_ID=your_razorpay_key
```

---

# ▶️ Run the Project

## Start Backend

```bash
cd virtual-ui-server
npm run dev
```

## Start Frontend

```bash
cd virtual-ui-client
npm run dev
```

## Run UI Library

```bash
cd virtual-ui-lib
npm run dev
```

---

# 📦 Install NPM Package

```bash
npm install virtual-ui-components
```
